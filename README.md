## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/kenprogrammer/daraja-sdk-php

# Access
$ cd daraja-sdk-php

# Install dependencies
$ composer install

# Run the project
$ php -S localhost:8080

# The server will initialize in the <http://localhost:8080>
```
